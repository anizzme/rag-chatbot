# Multi-Document RAG Chatbot

## 📌 Description
This project implements a Retrieval-Augmented Generation (RAG) system that allows users to query multiple documents and receive context-aware responses using LLMs.

## 🚀 Features
- Document ingestion and chunking
- Embedding-based semantic search
- Context-aware response generation
- Multi-document querying support

## 🛠️ Tech Stack
- Python
- LangChain
- OpenAI API
- ChromaDB / FAISS

## ⚙️ How It Works
1. Documents are split into smaller chunks  
2. Each chunk is converted into embeddings  
3. Relevant chunks are retrieved based on user query  
4. LLM generates response using retrieved context  

## ▶️ How to Run
1. Install dependencies  
2. Add your API key  
3. Run the script  

## 📊 Key Highlights
- Improved response relevance using semantic search  
- Efficient document retrieval using vector database
